# 🦄 Unicorn Magic Calculator

A fun and colorful calculator built using **HTML, CSS, and JavaScript**. This project combines basic calculator functionality with a magical unicorn-themed user interface, floating star animations, sound effects, and interactive visual effects.

## ✨ Features

* Basic arithmetic operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)

* Beautiful unicorn-themed design

* Animated floating stars background

* Interactive button hover effects

* Sound effects on button clicks

* Success and error sound notifications

* Celebration animation after successful calculations

* Responsive and beginner-friendly code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Unicorn-Magic-Calculator/
│
├── index.html
├── style.css
├── script.js
│
└── Image/
    ├── unicorn.png
    └── click.wav
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Make sure the image and sound files are inside the **Image** folder.
4. Open `index.html` in your browser.
